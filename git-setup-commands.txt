# Complete Git Setup Commands for Satyam Foundation Website

# Step 1: Initialize Git repository
git init

# Step 2: Add ALL files (not just README.md)
git add .

# Step 3: Commit all files
git commit -m "Initial commit - Satyam Foundation E-commerce Website"

# Step 4: Set up main branch
git branch -M main

# Step 5: Add your GitHub repository
git remote add origin https://github.com/rickyjones544-debug/satyam-foundation-website.git

# Step 6: Push to GitHub
git push -u origin main

# After this, your website will be ready for Vercel deployment!
